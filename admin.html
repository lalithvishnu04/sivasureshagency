Category Tree & Navigation

One tree runs the whole site. Build it in three levels — Main Heading (menu column) → Main Category (e.g. Male Doctor Uniform) → Sub Category (e.g. Full Sleeve). This powers the header dropdown, the shop filters, and where products are filed. Mark a heading Signature to show it as a separate highlighted collection (like CliniFlex). Click Save & Publish to push everything live.

Add Main Heading

Category Tree Heading → Main Category → Sub Category

Rename anything inline. Under each Heading add Main Categories; under each Main Category add Sub Categories. Products are filed against a Main Category (and optionally a Sub Category). Menu thumbnails are pulled automatically from your product images.
